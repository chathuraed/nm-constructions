### Hero Section (`/images/hero/`)
- **hero-construction.jpg** - Main hero image (1920x1080px recommended)
  - Suggested: Construction site, modern house, or landscaped property
- **hero-landscaping.jpg** - Alternative hero image

## Image Guidelines

1. **Format**: JPG for photos, PNG for logos with transparency
2. **Size**: Optimize for web (compress to <500KB per image)
3. **Dimensions**: 
   - Hero images: 1920x1080px
   - Project thumbnails: 800x600px
   - Service icons: 400x400px
4. **Quality**: High quality, professional photography
5. **Content**: Real construction/landscaping photos (no stock cartoons)

## Temporary Placeholders

Until you add actual images, the website uses:
- Gradient backgrounds with icons/emojis
- Placeholder text indicators
- Unsplash API for demo purposes (optional)

## How to Add Images

1. Add your image files to the appropriate folders above
2. Keep the exact filenames as listed (or update the image imports in components)
3. The Next.js Image component will automatically optimize them
